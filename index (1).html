🌐 Modo Online (tempo real)

Conecte a TV e os celulares à mesma sala. Você atualiza de qualquer dispositivo e a TV reflete na hora. Os dados ficam no seu Supabase (grátis).

Use o mesmo código em todos os dispositivos do mesmo torneio.
Cole o endereço do projeto. Se vier com /rest/v1/ ou outro caminho no fim, o sistema remove sozinho.
Use a chave publishable (começa com sb_publishable_) ou a anon public antiga (começa com eyJ). Ambas podem ficar no dispositivo. Nunca use uma chave secreta (sb_secret_ ou service_role).
Como obter esses dados (1ª vez, ~5 min)
  1. Crie conta grátis em supabase.com → New Project.
  2. No projeto: Project Settings → API. Copie o Project URL e a chave anon public.
  3. Vá em SQL Editor e rode o script que está no arquivo de instruções (cria a tabela tournaments).
  4. Cole URL e chave aqui, escolha a função e salve.
